## v1.1.1 - Desktop and Build Update

This release focuses on the desktop experience, interface behavior, and project packaging. It improves the window manager, expands the built-in UI flow, and aligns the build output with the `elevn32` project name.

### Highlights

- Improved the desktop environment and window manager behavior
- Expanded desktop-side shell integration inside the terminal window
- Refined the boot screen and visual presentation
- Added theme-aware UI behavior, including dark-mode support
- Updated framebuffer drawing paths for cleaner rendering behavior
- Renamed the generated ISO image to `elevn32.iso`
- Removed old logo and asset-conversion pipeline files that were no longer needed

### Included Desktop Apps

- Terminal
- Calculator
- Paint
- Clock
- Settings
- Notepad
- System Info
- Snake
- Minesweeper
- Files

### Notes

This release is mainly a desktop-facing polish and packaging update rather than a low-level kernel architecture rewrite. It builds on the earlier kernel foundation and pushes the project toward a more complete and usable graphical environment.
